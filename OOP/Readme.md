## Java ##

1. *Object-Oriented Programming* 
   - Java is a **purely object-oriented** programming language.  
   - Everything in Java is defined inside a class.  

2. *Statically Typed Language*
   - Java is a statically typed language, meaning types are checked at compile time.  
   - Using undefined or mismatched types results in errors.  

3. *Runtime Environment*  
   - Java requires the **Java Runtime Environment (JRE)** to run Java-based applications.  
   - JRE includes the Java Virtual Machine (JVM), which executes Java bytecode.  

4. *Compiler*  
   - Java uses a **compiler** (javac) to convert source code into bytecode.  
   - Bytecode is then interpreted by the JVM to establish communication between the user and the machine.  

5. *Platform Independence*  
   - Java’s “Write Once, Run Anywhere” feature makes it platform-independent.  

## Key Principles of Object-Oriented Programming in Java ##
**Encapsulation** : Wrapping data (fields) and methods into a single unit (class).
**Inheritance** : A mechanism to derive new classes (child classes) from existing classes (parent classes).
**Polymorphism** : The ability of a single interface to represent different types of behavior (method overriding).
**Abstraction** : Hiding implementation details and exposing only necessary features through abstract classes and interfaces.
